Safari

Safari est un navigateur web pour macOS, iPadOS et iOS développé par Apple, dont le moteur de rendu HTML WebKit est fondé sur KHTML.

Il est téléchargeable gratuitement depuis le 7 janvier 2003, soit depuis Mac OS X v10.2. Le 8 juin 2009 la version 4.0 est sortie pour les plateformes Mac OS X v10.4 (ou ultérieur), Windows XP et Vista (ou ultérieur). Ce navigateur est celui installé par défaut sur tous les ordinateurs Mac depuis Mac OS X v10.3. Pour le fonctionnement de Safari 4.0, il faut que Quartz Extrême soit pris en charge par la carte graphique. La version pour Microsoft Windows a été abandonnée : la dernière, Safari 5.1.7, est sortie en mai 2012.
