Syntaxe HTML

Le code de cette page comporte plusieurs erreurs de syntaxe HTML.

Pouvez vous en trouver 5 ?

Erreurs dans le code :

  • Erreur #1 : la valeur de l'attribut id ne peut pas être une liste : id="title main-title".
  • Erreur #2 : la balise <text> n'est pas un élément HTML valide. Il faut utiliser <p> ou simplement laisser le texte.
  • Erreur #3 : la valeur de l'attribut id est unique les éléments <h1> et <h3> ne peuvent pas avoir le même id.
  • Erreur #4 : l'élément <li>(List Element) doit être enfant d'un élément <ul> (Unordered List) ou <ol> (Ordered List).
  • Erreur #5 : l'élément <div> doit avoir une balise fermante </div>. Ceci génère un 2e élément <div> vide dans la page.

Note : dans la ligne suivante : <section id="description" class="description text"> la valeur "description" est toute à fait valide pour les attributs id et class en même temps.
